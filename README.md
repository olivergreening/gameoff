## Controlsmp:

Use the `Arrow Keys` or `WASD` keys to switch lanes and accelerate

## Development

**Requirements**

- [Node.js 20+](https://nodejs.org)
- [npm](https://www.npmjs.com/package/npm) or [Yarn](https://yarnpkg.com/)
- [Git LFS](https://git-lfs.github.com/)

To run a dev server:

    npm start

To produce a build:

    npm build
